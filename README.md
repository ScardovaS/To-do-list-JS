# To-Do List em JavaScript

Projeto simples de lista de tarefas feito com:
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Adicionar tarefas
- Remover tarefas
- Marcar como concluída
- Filtros (todas,pendentes,concluídas)
- Persistência com localStorage

## Objetivo
Praticar organização de código e manipulação do DOM.

## Aprendizados

- Eventos
- Arrays e objetos
- LocalStorage
- Lógica dos filtros
